# 🌊 TERRA: Restoration of the Deep

**Vibe Jam 2026 Submission**

Terra is a high-fidelity, top-down exploration and survival game where you take on the role of a lone operative tasked with restoring a dying planet. Journey from the toxic depths of the ocean to the lush, supercharged surface garden.

---

## 🚀 The Mission

The planet is suffocating under layers of industrial pollution and toxic waste. Your mission is to descend into the depths, purify the waters, and eventually reach the surface to restore the ancient garden.

### 🎮 Gameplay Features
*   **Evolutionary Progression:** Start as a deep-sea **Diver** and evolve into the **Iron Guardian**—a supercharged adventurer equipped with an integrated Red Arc Reactor.
*   **Three Dynamic Layers:** 
    *   **Layer 1: The Ocean Floor** - Navigate the debris and clear the initial toxicity.
    *   **Layer 2: The Deep** - Face more dangerous pollutants in the lightless abyss.
    *   **Layer 3: The Surface Garden** - Reach the final stage, jump across restored land, and blast the remaining "Smog" enemies.
*   **Advanced Combat:** Swap your standard blaster for the **Red Uni-Beam**—a powerful chest-mounted laser that fires directly from your Arc Reactor.
*   **Restoration System:** Clean tiles to restore the environment. Watch as seaweed, corals, and butterflies return to the world in real-time.

---

## 🛠️ Tech Stack
*   **Engine:** [Three.js](https://threejs.org/) (WebGL)
*   **Build Tool:** [Vite](https://vitejs.dev/)
*   **Language:** Vanilla JavaScript / ES6+
*   **Aesthetics:** High-contrast Midnight Glass UI, procedural environment generation, and skeletal-based character animations.

## ⌨️ Controls

| Key | Action |
| --- | --- |
| **W, A, S, D** | Move Character |
| **Mouse Move** | Aim Uni-Beam / Blaster |
| **Left Click** | Fire Laser / Blaster |
| **Spacebar** | **JUMP** (Surface Layer Only) |
| **C** | Open Armory / Crafting Menu |
| **T** | Toggle Torch (Underwater Only) |
| **1 - 4** | Express Emotes (Happy, Angry, Wave, Heart) |

## 🏗️ Installation & Setup

1. **Clone the repository:**

   ```bash

   ```
3. **Launch the mission:**
